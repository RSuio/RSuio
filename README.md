## 🚀 Projeto em Destaque

### 🏋️ ForgeFit — Sistema de Gestão de Academia Digital

> Projeto ponta a ponta que elimina o papel no ambiente das academias. Envolveu desde o planejamento do produto até a implementação do back-end e mobile.

**Visão de Produto & UX:**
* Idealização com **Matriz CSD** e Mapas de Empatia.
* Prototipagem completa de interfaces utilizando **Figma**.

**O que foi construído (Técnico):**
* ⚙️ **API REST** com FastAPI, autenticação JWT e rotas seguras.
* 🗄️ **Banco de dados** relacional (SQLAlchemy + SQLite) com migrações gerenciadas pelo Alembic.
* 📱 **App Mobile** nativo com Kotlin + Jetpack Compose.
* 📐 Aplicação de **Padrões de Projeto** (Strategy, Template Method) para um código limpo e escalável.
* ✅ **Testes Unitários** utilizando pytest para garantir a estabilidade da aplicação.
